# Port Status Checker

## Description

A Python-based Port Status Checker that uses socket programming to determine whether a specified port on a target system is open or closed.

## Features

* Checks the status of a user-specified port.
* Uses Python's built-in socket library.
* Displays whether the port is OPEN or CLOSED.
* Simple command-line interface.

## Requirements

* Python 3.x

## How to Run

1. Clone the repository:

   ```
   git clone <repository-link>
   ```

2. Navigate to the project directory:

   ```
   cd port_status_checker
   ```

3. Run the script:

   ```
   python port_status_checker.py
   ```

4. Enter the target host and port number when prompted.

## Example

Input:

```
Enter target host (e.g., localhost): localhost
Enter port number: 80
```

Output:

```
Port 80 is OPEN
```
